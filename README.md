# College ERP Frontend (React + Vite)

## Run
1. Install dependencies:
   - `npm install`
2. Copy env:
   - copy `.env.example` to `.env`
3. Start dev server:
   - `npm run dev`

## Environment
- `VITE_API_BASE_URL` defaults to `http://127.0.0.1:8000/api`

## Features
- JWT login page
- Protected routes
- Role-aware dashboard summary cards
- API integration with backend modules
